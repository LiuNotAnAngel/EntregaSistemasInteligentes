# EntregaSistemasInteligentes

Este repositorio contiene la entrega de la asignatura de Sistemas Inteligentes.

## Descripción

El proyecto consiste en una serie de cuadernos Jupyter (`.ipynb`) pensados para ejecutarse en Google Colab, donde se desarrollarán y probarán algoritmos de inteligencia artificial aplicados a distintos problemas.

## Requisitos

- Python 3.x
- Acceso a Google Colab
- Las dependencias específicas se incluirán en cada cuaderno, según corresponda.

## Estructura del repositorio

- `main.ipynb`: Cuaderno principal con la solución base.
- Otros cuadernos pueden añadirse a medida que avance el proyecto.
